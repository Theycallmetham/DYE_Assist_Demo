DYE Assist Demo Kit

What’s included
- DYE_Assist_Demo.html — standalone clickable demo
- DYE_Assist_Pitch_Deck.pptx — pitch deck
- Demo_Walkthrough.txt — short live demo script

How to use
1. Open DYE_Assist_Demo.html in any browser.
2. Best phone demo:
   - Upload the HTML to GitHub Pages, Dropbox, Google Drive, or iCloud Drive.
   - Open the shared link on your phone in Safari or Chrome.
3. Best in-person pitch flow:
   - Open the pitch deck first.
   - Then switch to the demo and walk through:
     Home → Smart Diagnosis → Diagnosis Result → Scan Marker Live → Parts → Manual

Notes
- This is a front-end concept demo, not a production app.
- External buttons open current DYE links for manuals and parts.
- Marker images are pulled from DYE product image URLs.

Suggested live story
1. “This is a support and sales tool, not just an app.”
2. “A user starts with their marker, symptom, or camera scan.”
3. “The app recommends the likely fix, shows where the O-ring goes, and links to the exact DYE resources.”
4. “That reduces support friction and creates more direct parts-store conversion.”
